# C2C 이커머스 웹사이트에서의 고립 페이지 문제

이 블로그 글에서는 제가 근무 중인 C2C 이커머스 플랫폼에서 SEO **고립 페이지** 문제를 해결한 경험에 대해 이야기하려고 합니다. 이 문제는 회사 웹사이트의 SEO를 개선하기 위해 해결해야 할 가장 중요한 문제 중 하나였음을 강조하고 싶습니다.

SEO에서의 **고립 페이지**는 웹사이트 내에서 다른 페이지와 연결이 전혀 없는 페이지를 의미합니다. 이러한 문제는 페이지가 생성된 후 잊혀지거나, 해당 페이지로의 링크가 제거되어 검색 엔진이 해당 페이지를 발견하고 크롤링하기 어렵게 만들 때 발생할 수 있습니다. 고립 페이지는 검색 엔진이 웹사이트를 완전히 크롤링하고 인덱싱하지 못하게 하여 웹사이트의 SEO에 부정적인 영향을 미칠 수 있습니다.

## 고립 페이지를 식별하는 방법

1. **Google Search Console 사용**: Google Search Console은 사이트의 모든 페이지가 크롤링 및 인덱싱된 보고서를 제공합니다. 이 보고서를 사이트의 사이트맵과 비교하면 누락된 페이지를 쉽게 식별할 수 있으며, 이 페이지들이 바로 고립 페이지일 수 있습니다.
2. **Ahrefs 또는 Screaming Frog와 같은 SEO 도구 사용**: 특히 Screaming Frog는 무료로 사용할 수 있으며, 웹사이트 크롤러로서 고립 페이지를 식별하는 데 도움이 됩니다. 웹사이트 URL을 입력하면 이 도구가 사이트를 크롤링하고 모든 페이지를 포함한 보고서를 생성하여 고립 페이지를 찾아줍니다.
3. **사이트맵 확인**: 사이트맵은 사이트의 모든 페이지 목록을 포함합니다. 만약 사이트맵에 포함되지 않은 페이지가 있다면, 그것은 고립 페이지일 가능성이 높습니다.

## 고립 페이지를 해결할 수 있을까요?

그건 상황에 따라 다릅니다. 만약 모든 사용자의 제품 페이지가 삭제된 후에도 SERP(검색 엔진 결과 페이지)에 인덱싱되고 표시되기를 원한다면, 답은 "아니오"입니다. 제가 찾은 유일한 방법은 고립 페이지의 수를 줄이는 것뿐이었습니다.

우리는 C2C 플랫폼이었기 때문에, 사용자들이 제품을 생성, 수정, 삭제하는 경우가 많았습니다. 제품 페이지로의 링크를 보여주는 페이지는 고정되지 않았으며, 페이지에 표시된 링크는 항상 업데이트되었고, 이로 인해 링크가 손실되고 크롤러가 사이트맵에 있는 링크를 찾지 못하게 되었습니다.

그러나, 사용자가 찾고자 하지 않는 페이지를 삭제한다면 문제를 해결할 수 있습니다. 따라서, 사이트맵에 인덱싱하고 검색할 가치가 있는 페이지들만 추가할 것을 권장합니다.

## 고립 페이지를 해결하는 방법

1. **내부 링크 추가**: 고립 페이지를 해결하는 가장 쉬운 방법은 내부 링크를 추가하는 것입니다. 이를 통해 검색 엔진이 페이지를 크롤링하고 인덱싱할 수 있게 되며, 방문자들에게도 더 나은 사용자 경험을 제공할 수 있습니다.

- 세부적으로 다음과 같은 작업을 수행할 수 있습니다:
  1. 카테고리 페이지를 생성하고 해당 페이지들 간 내부 링크를 만듭니다.
  2. 페이지네이션(pagination)과 제품 페이지로의 링크를 추가하여 도움을 줍니다.
  3. 선택적으로, 랜딩 페이지에 인기 있는 제품 섹션을 만들어 많은 사용자가 검색할 제품 페이지로 내부 링크를 만듭니다.

2. **관련 페이지로 리디렉션**: 더 이상 관련성이 없는 고립 페이지는 사이트 내의 관련 페이지로 리디렉션할 수 있습니다. 이를 통해 사용자가 링크가 없는 페이지로 이동하는 것을 방지하고, 고립 페이지의 링크 자산(link equity)이 관련 페이지로 전달되도록 할 수 있습니다.
3. **페이지 삭제**: 더 이상 관련성이 없는 고립 페이지가 리디렉션될 수 없다면, 해당 페이지를 삭제하는 것이 좋습니다. 이를 통해 사용자가 링크가 없는 페이지로 이동하는 것을 방지하고, 검색 엔진이 가치 없는 페이지를 크롤링하지 않도록 할 수 있습니다.
4. **최적화된 논리적 URL 구조 사용**: Google의 유명한 Search Advocate인 John Mueller가 다음과 같이 말했습니다:

> "_대부분의 경우, 우리는 URL을 콘텐츠의 식별자로 취급합니다._"
>
> - John Mueller, Google Search Advocate

## 마무리

**SEO**는 정말 고통스럽습니다. 왜냐하면 항상 뭔가 잘못되었다고 하고, 검색 엔진은 항상 업데이트되기 때문입니다. SEO 작업은 끊임없이 해야 합니다.

고립 페이지를 식별하고 해결함으로써 웹사이트의 SEO를 개선하고 방문자들에게 더 나은 사용자 경험을 제공할 수 있습니다. 고립 페이지를 식별하고 해결하는 방법이 확실하지 않다면, 웹사이트를 최적화하고 검색 엔진 순위를 개선하는 데 도움을 줄 수 있는 SEO 전문가와 협력하는 것을 고려해 보세요.
